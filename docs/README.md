# AHS Ecosystem - Official Documentation

**Authority of Truth**

This directory (`/docs`) serves as the ultimate Source of Truth for the entire AHS Ecosystem project. While technical constraints, operational tools, and agent skills (such as those in `.agent/`) govern the automated execution, this documentation directory is the **authoritative reference** for humans, architects, auditors, and stakeholders.

## Structure
- `/architecture`: Contains Constitutional documents, System Architecture Documents (SAD), and Architecture Decision Records (ADRs).
- `/architecture/adr`: Historical and active decisions binding to all Cells (e.g. ADR-009, ADR-010).
- `/architecture/constitution`: Core principles and global invariants (e.g. Blueprint V3.1).
- `/product`: Product Requirements Documents (PRDs) containing KPIs, Personas, and Roadmaps.

Any deviation in code that contradicts the documents within this directory is considered a bug or an alignment drift and must be addressed.
